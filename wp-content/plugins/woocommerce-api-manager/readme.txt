== Upgrade Notice ==

= 2.0.0 =

Version 2.0 is a major release. Make a full database backup before upgrading. Make a full site backup, update your theme and extensions, and [review update best practices](https://docs.woocommerce.com/document/how-to-update-your-site) before upgrading. Install version 2.0 on a test site to confirm the database migration process, and functionality works with your site, other plugins, and themes. For more information, refer to the [documentation](https://docs.woocommerce.com/document/woocommerce-api-manager/).
